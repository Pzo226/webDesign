Definition of Terms

Domain name

It is a special name that identifies a website. It usually contains the chosen name for your website with name extensions such as .com or .net

Web Hosting

A place or a server where your website content is saved in order to be accessible via the World Wide Web.

SEO (Search Engine Optimization)

A technique or process used to increase visitors in your website using a natural search engine results.

CMS (Content Management System)

It is a software or program used to manage the content of your website. It allows you to edit, publish and modify the content of your website.

Responsive Web Design

It is a kind of web design that allows users to access and view website compatible with the device they are using. This design has the ability to adjust depending on the user's device such as mobile phones, tablets, desktop PCs.
